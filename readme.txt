Upload zipped .txt files and insert them as wordpress posts

== Description ==
<pre>
Want to import lots of plain text files as wordpress posts?
You can easily achieve this with TxtAsPost Mod.
#1, Compressed all .txt files in zip.
#2, Choose the category you want them to be published in.
#3, Set the desired posting interval.
#4, Click upload button.
#5, Done.
If you need a customized version, please <a href="http://www.wordpressplugindeveloper.com/get-a-quote">get a quote here</a>. Note: it is not free.
</pre>
== Installation ==

1. Extract txtaspost.zip in the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "TXT As Post" In Setting
4. Follow the on-screen instructions
